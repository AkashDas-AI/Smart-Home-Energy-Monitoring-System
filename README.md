# Smart Home Energy Monitoring System

## 📌 Project Overview

The Smart Home Energy Monitoring System is an IoT-based platform designed to monitor, analyze, and visualize household electricity consumption in real time.

The project implements a dual-track architecture consisting of:

* An ESP32-based hardware implementation using voltage and current sensors
* A Python-based virtual smart meter simulator for hardware-independent testing and development

Both implementations follow the same telemetry pipeline and publish energy measurements through MQTT, enabling seamless integration with monitoring platforms and cloud dashboards.

The system calculates:

* Active Power
* Apparent Power
* Reactive Power
* Power Factor
* Energy Consumption
* Electricity Cost

It also detects abnormal electrical conditions such as:

* Overvoltage
* Undervoltage
* Overcurrent
* Overload
* Excessive Energy Consumption

Telemetry is transmitted through MQTT, visualized using Home Assistant and Grafana, and additionally published to ThingSpeak for cloud-based monitoring.

The project demonstrates real-world concepts used in modern:

* Smart Homes
* Energy Management Systems
* Building Automation
* Industrial Monitoring Platforms
* IoT Analytics Solutions

## ▶️ How to Run

### 1. Clone Repository

```bash
git clone <repository-url>
cd Smart-Home-Energy-Monitoring-System
```

---

### 2. Create Virtual Environment

```bash
python -m venv venv
```

Activate:

Windows:

```bash
venv\Scripts\activate
```

Linux / macOS:

```bash
source venv/bin/activate
```

---

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4. Configure Environment Variables

Create:

```text
.env
```

Update:

```env
MQTT_BROKER=localhost
MQTT_PORT=1883
MQTT_TOPIC=home/energy/node1

ELECTRICITY_RATE=8.50

SIMULATION_INTERVAL=1

THINGSPEAK_CHANNEL_ID=YOUR_CHANNEL_ID
THINGSPEAK_API_KEY=YOUR_API_KEY
```

---

### 5. Start Mosquitto MQTT Broker

```bash
mosquitto -v
```

---

### 6. Run Simulator

```bash
python -m simulator.main
```

The simulator will:

* Generate telemetry
* Calculate energy usage
* Estimate electricity costs
* Detect alerts
* Publish MQTT messages
* Update ThingSpeak
* Generate CSV logs
* Generate PDF reports
* Generate sample datasets
